# EasyConfigSingleBoard

【开发中】

这是一个用于单板计算机（如树莓派）的简易配置工具。

## 功能特性

- **混合WIFI模式 (AP + STA)**：主机同时作为热点并连接到其他WIFI。
- **设备状态监控**：通过Web页面查看CPU、内存、网络状态。
- **简易配置**：用户连接热点后即可通过网页配置主机的联网信息。

## 使用指南

### 1. 准备环境
确保您的单板计算机（如树莓派）使用的是 NetworkManager 管理网络（Raspberry Pi OS Bookworm 默认使用）。

### 2. 开启混合 WIFI 模式
运行配置脚本，这会创建一个名为 `EasyConfig-XXX` 的热点：
```bash
sudo ./setup_mixed_wifi.sh
```

### 3. 启动配置网页
运行 Web 服务：
```bash
./run.sh
```

### 4. 访问与配置
1. 使用手机或电脑连接热点 `EasyConfig-XXX` (默认密码 `12345678`)。
2. 打开浏览器访问 `http://10.42.0.1:5000`。
3. 在页面中可以查看 CPU、内存信息，并扫描周围 WIFI 进行连接。

## 注意事项
- 并不是所有的 WIFI 芯片都支持 AP + STA 同步运行，如果硬件不支持，开启热点后可能无法连接其他 WIFI。
- 配置脚本需要 `sudo` 权限来修改网络配置。
