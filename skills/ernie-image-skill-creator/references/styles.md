# Visual Style Guide for Image Generation Skills

## Modern/Minimalist Style

### Characteristics
- Clean, simple lines
- Lots of white space
- Sans-serif typography
- Monochromatic or limited color palette
- Geometric shapes and forms
- No unnecessary decorations

### Color Palette
- Primary: #FFFFFF (white), #000000 (black)
- Accent: #F5F5F5 (light gray), #333333 (dark gray)
- Highlight: #007AFF (blue), #FF3B30 (red) - optional

### Typography
- Primary: Helvetica, Arial, Roboto
- Secondary: Open Sans, Lato, Montserrat
- Size: Large titles, medium body text

### Layout Principles
- Grid-based layout
- Generous margins and padding
- Clear visual hierarchy
- Minimal element count

### Prompt Keywords
```
minimalist, clean, simple, white space, sans-serif, geometric, modern, contemporary
```

## Rustic/Vintage Style

### Characteristics
- Natural, organic textures
- Handwritten or script typography
- Warm, earthy color palette
- Aged, weathered appearance
- Artisan, handcrafted feel
- Vintage patterns and motifs

### Color Palette
- Primary: #F5F5DC (beige), #DEB887 (burlywood)
- Secondary: #8B4513 (saddle brown), #556B2F (dark olive green)
- Accent: #CD853F (peru), #A0522D (sienna)

### Typography
- Primary: Script fonts (Pacifico, Dancing Script)
- Secondary: Serif fonts (Georgia, Times New Roman)
- Handwritten: Caveat, Indie Flower

### Layout Principles
- Asymmetric, organic layouts
- Natural flow and movement
- Texture overlays and backgrounds
- Decorative borders and frames

### Prompt Keywords
```
rustic, vintage, natural, organic, textured, handwritten, aged, artisan, handmade
```

## Elegant/Luxury Style

### Characteristics
- Sophisticated, refined appearance
- High-quality materials and finishes
- Metallic accents (gold, silver, copper)
- Serif typography with perfect kerning
- Subtle gradients and shadows
- Premium, high-end feel

### Color Palette
- Primary: #FFFFFF (white), #000000 (black)
- Metallic: #FFD700 (gold), #C0C0C0 (silver), #B87333 (copper)
- Accent: #1A1A1A (jet black), #F8F8FF (ghost white)

### Typography
- Primary: Serif fonts (Garamond, Bodoni, Didot)
- Secondary: Elegant sans-serif (Optima, Avenir)
- Display: Decorative scripts for titles

### Layout Principles
- Symmetrical, balanced layouts
- Generous white space
- Perfect alignment and spacing
- Subtle, refined details

### Prompt Keywords
```
elegant, luxury, sophisticated, refined, premium, high-end, metallic, gold, silver
```

## Style Mixing Guidelines

### Combining Styles
When combining styles, consider:
1. **Compatibility**: Do the styles complement each other?
2. **Balance**: Is one style dominant and another secondary?
3. **Cohesion**: Do they create a unified visual language?
4. **Purpose**: Does the combination serve the user's goal?

### Example Combinations
- **Modern + Elegant**: Clean lines with luxurious materials
- **Rustic + Playful**: Natural textures with fun colors
- **Professional + Creative**: Structured layout with artistic elements
- **Futuristic + Natural**: High-tech with organic elements

### Prompt Structure for Mixed Styles
```
[Primary style] design with [secondary style] elements, 
[specific characteristics from primary], [specific characteristics from secondary], 
[unifying elements], [desired outcome]
```

## Style Application Process

### Step 1: Identify Core Requirements
- What is the purpose of the image?
- Who is the target audience?
- What message should it convey?
- What emotions should it evoke?

### Step 2: Select Primary Style
Choose the main style that best fits the requirements.

### Step 3: Add Supporting Elements
Consider adding elements from other styles to enhance the design.

### Step 4: Define Specific Parameters
- Color palette
- Typography
- Layout structure
- Visual elements

### Step 5: Create Prompt
Combine all elements into a comprehensive prompt.

### Step 6: Test and Refine
Generate images and refine the prompt based on results.
