## Prompt Optimization Tips

1. **Be Specific**: Include exact dimensions, positions, and styles
2. **Use Adjectives**: Describe the visual qualities you want
3. **Reference Styles**: Mention specific design styles or periods
4. **Include Technical Details**: DPI, resolution, file format
5. **Specify Colors**: Use specific color names or hex codes
6. **Define Layout**: Describe the exact arrangement of elements
7. **Consider Readability**: Ensure text will be legible
8. **Test Variations**: Try different prompt versions to find the best
