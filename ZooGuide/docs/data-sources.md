# 数据来源说明

红山森林动物园的场馆数据从以下公开资料整理：

1. **南京红山森林动物园官网**：https://www.njhszoo.com/
2. **百度百科 - 南京红山森林动物园**：https://baike.baidu.com/item/南京红山森林动物园
3. **小红书 / 公众号** 公开的游客评价（用于补充场馆标签、人流特征）

## 数据组织原则

- 每个场馆独立条目，包含位置、动物、标签、推荐时长
- 步行时间根据百度百科提到的「北门距离大熊猫馆 300 米」「南门新区游览线 1.6 公里」等公开数据估算
- 标签如「亲子」「网红」「明星动物」综合自红山的微信公众号推文（饲养员日常记录）与游客反馈

## 已知的数据局限

- 步行时间矩阵是估算值，未实地测量
- 部分场馆的「推荐参观时长」参考官方说明但加入了保守缓冲
- 动物列表可能与实时有出入（部分动物已离世、转赠等）
- 2025 年 10 月开放的南门新区信息可能持续更新

## 后续数据更新

如果项目要正式落地，建议：
1. 与红山动物园合作获取官方讲解词与动物资料
2. 实地测量步行时间
3. 接入实时客流数据（红山有官方小程序）
