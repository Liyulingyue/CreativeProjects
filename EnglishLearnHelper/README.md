# EnglishLearnHelper

英语学习辅助器

## 快速开始

### 1. 克隆项目

```bash
git clone https://github.com/your-repo/EnglishLearnHelper.git
cd EnglishLearnHelper
```

### 2. 获取单词数据

```bash
# 创建 Data 目录并克隆单词库
mkdir Data
cd Data
git clone https://github.com/fanhongtao/IELTS.git
cd ..
```

### 3. 启动后端

```bash
cd backend
pip install -r requirements.txt
python run.py
```

### 4. 启动前端

```bash
cd frontend
npm install
npm run dev
```

## 功能

- 单词查询
- 翻译
- 语法检查
- AI 对话练习
